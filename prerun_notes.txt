SECTION 2: INSTANCE INITIALIZATION (ON AWS/LAMBDA)
Bash
# 1. System Check
nvidia-smi  # Confirm H100 80GB HBM3

# 2. Repo Setup
git clone https://github.com/pauliano22/triton-gpu-kernels.git
cd triton-gpu-kernels

# 3. Environment Setup
python3 -m venv venv
source venv/bin/activate
pip install --upgrade pip
pip install torch triton matplotlib pandas
pip install -e .

# 4. Hardware Optimization (Requires Sudo)
sudo nvidia-smi -pm 1        # Enable Persistence Mode
sudo nvidia-smi -lgc 1755    # Lock Graphics Clock to Max (Hopper)

SECTION 5: CRITICAL TROUBLESHOOTING
"Out of Memory (OOM)": If FlashAttention fails, reduce BATCH or SEQ_LEN in benchmarks/bench_flash_attn.py.

"ModuleNotFoundError": Ensure you ran pip install -e . in the root directory.

"Driver too old": AWS P5 instances should have CUDA 12.x. If not, you may need a different AMI (Amazon Machine Image).

"Permission Denied": Use sudo for nvidia-smi commands.

SECTION 6: THE SHUTDOWN
TERMINATE the instance via the web console.

STOPPING is not enough; AWS still charges for the provisioned EBS storage. DELETE the instance to stop all charges.
